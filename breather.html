WHAT I DID

I developed 'Breather', a website that is inspired by the yogic breathing technique of Pranayama.Breather aims to let users perform the 4-7-8 Pranayama that is known to calm down your mind and help you relax.

The website is designed to be very simple and easy to use. The user can start the timer and follow the instructions to perform the breathing technique. The website will guide the user through the process and will alert the user when to inhale, hold and exhale.

I also added a set of tools to compliment my website as a complete self help toolpack out there with 'Breather Labs' which includes a Focus Timer with Lo-Fi background music and Meditation with Nature Sounds.

Breather has a beautiful and calming UI with a soothing color scheme and a minimalistic design that helps the user to focus on the breathing technique. It is made to let users relax and take a break from their busy life.
