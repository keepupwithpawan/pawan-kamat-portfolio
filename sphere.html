WHAT I DID

These are two of my 3D models, a stress ball and a keyboard. Stress Ball is one of my earliest projects developed when I was exploring 3D deisgn using Spline. It was developed as a tool that mimics the functioning of a Stress Ball that peoplewith short temper, anxiety use while driving.

It is a 3D sphere that can be rotated using the mouse and expands many smallers spheres from the center to out to give a feel of a stress ball. The project was developed using HTML, CSS and Javascript. The 3D sphere was developed using Spline 3D.

Keyboard is a small 3D model of a keyboard that was developed using Spline 3D. The keyboard reacts to key presses and displays the status of the key being pressed live on the web.
