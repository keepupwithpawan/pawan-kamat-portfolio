WHAT I DID

My Space was developed as a part of my internship at Anzen Technologies Pvt. Ltd. My task was to develop the landing page for an Ecommerce website that will basically serve as a testing background for the 'Add to Cart' feature.

My main task was to develop the 'Add-to-Cart' button that dynamically updated the count of the cart items if user adds or deletes items. I cannot share the code for the project as it is confidential.

This project served as a training for me to explore AJAX and dynamic requests as this was developed by the end of my second year and I was still pretty new to backend development.
